## 🎯 What is f_clean?

*f_clean* is a starter app built with Flutter technology. It uses clean architecture, which means the design helps keep the app organized and easy to expand.

This app can serve as a base for simple projects, especially if you want to try Flutter apps without building from scratch. It has the main parts developers use, like routing, state management, and network functions.

## 🗂️ Understanding the App Structure

Inside the app files, you will find several folders and files that control different parts of the app.

- `lib/`: Main folder where most files live.
  - `bloc/`: Handles app state and logic.
  - `data/`: Deals with saving and loading data.
  - `routes/`: Contains paths to different app screens.
  - `ui/`: Layout and design code for the screens.
- `assets/`: Images and style files.
- `docs/`: Information and user guides.
